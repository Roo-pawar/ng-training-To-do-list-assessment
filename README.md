# Todo Application (Angular)

## Features
- Add Task
- Edit Task
- Delete Task with confirmation
- Task Form Validation
- Professional UI

## Technologies Used
- Angular (Standalone Components)
- TypeScript
- HTML & CSS

## How to Run Project

1. npm install
2. ng serve
3. Open http://localhost:4200/
